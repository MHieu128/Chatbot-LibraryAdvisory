Welcome to Library Advisory System

This system helps you evaluate and compare programming libraries and frameworks for your projects.

Key Features:
  • Library Analysis: Get detailed technical evaluations of any library
  • Side-by-side Comparison: Compare two libraries with comprehensive analysis
  • Smart Recommendations: Get personalized suggestions based on your requirements
  • Risk Assessment: Understand security, maintenance, and business risks
  • Cost Analysis: Evaluate licensing costs and total cost of ownership
    • {% if ai_enabled %}
  • AI-Powered Insights: Get intelligent analysis using Azure OpenAI
    • {% endif %}
Quick Start:
  1. Go to the Chat page to start asking questions
  2. Try typing "analyze React" or "compare Vue vs Angular"
  3. Use the Compare page for detailed library comparisons
  4. Visit Recommendations to find libraries for specific needs
Available Commands
Analysis Commands
analyze <library>

Get detailed analysis of a specific library

compare <lib1> vs <lib2>

Compare two libraries side by side

recommend <category>

Get recommendations for a specific category

Utility Commands
list

Show all known libraries in the database

save

Save conversation to a markdown file

help

Display help information

The system provides analysis based on general knowledge and patterns. Always verify current information and test libraries in your specific environment before making decisions.
